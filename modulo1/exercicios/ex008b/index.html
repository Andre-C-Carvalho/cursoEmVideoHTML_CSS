Código Fonte

A tag CODE serve para que o HTML use uma fonte monoespaçada qur garante uma melhor leitura do código.

Exemplo1: window.document.getElementById('msg') formatação normal.

Exemplo2: window.document.getElementById('msg') formatação para código fonte.
