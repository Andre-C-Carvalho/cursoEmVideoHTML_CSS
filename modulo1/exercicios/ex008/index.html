Principais Formatações

Negrito / Destaque

Esta frase é exemplo do negrito utilizando a tag "b" (tag não semântica).

Já nesse caso, utiliza-se a tag STRONG para dar destaque ao termo importante da frase (tag semântica).

Itálico / Ênfase

Nesta frase utiliza-se a tag U para colocar o termo importante em itálico. (tag não semântica)

Nesta outra frase a tag EM é utilizada para dar ênfase ao termo importante. (tag semântica)

Texto Sublinhado / Texto Inserido

Esta frase é um exemplo do texto sublinhado usando a tag U. (tag não semântica)

Neste outro parágrafo utiliza-se a tag INS que dá o efeito sublinhado mas também tem o significado de um texto inserido a posteriori. (tag semântica)

Texto Riscado / Texto Deletado

Está frase é um exemplo de texto riscado, usando a tag STRIKE (tag não semântica). O texto aparece riscado mas não há qualquer significado para a tag.

A forma correta é dar o efeito de texto riscado utilizando a tag DEL (tag semântica). Essa tag tem o significado de um termo que foi suprimido ou que deve ser desconsiderado pelo leitor.

Texto Marcado

É possível criar um efeito tipo marca texto utilizado a tag MARK.

Sobrescrito / Subscrito

É possível utilizar o sobrescrito com a tag SUP para escrever coisas como a potenciação em equações matemáticas: x8 + x5 + x3 + x + 5 = 0

Também é possível usar o subscrito com a tag SUB para escrever coisas como fórmulas químicas: H2O, H2SO4, Ca(OH)2

Texto Grande / Texto Pequeno

É possível modificar o tamano da letra com a tag BIG: tamanho grande, tamanho normal

É possível modificar o tamano da letra com a tag SMALL: tamanho pequeno, tamanho normal
