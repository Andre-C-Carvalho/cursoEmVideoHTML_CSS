Links Externos / Internos

Para criar um hyperlink utilizamos a tag <a></a> que vem do inglês "anchor" (âncora). O principal parâmetro dessa tag é o href. Esse atributo é a URL completa do local para onde o usuário será direcionado caso interaja com o link externo. Segue um exemplo abaixo:

Otros dois parâmetrs importantes na tag A são o REL e o TARGET

  • target - tem as seguintes opções:
    • _blank - abre o link em uma nova aba do navegador, melhor opção para links externos;
    • _self - opcção padrão, abre o link na janela em que o usuário está. Não é interessante para links externos, retira o usuário do seu site.
  • rel - faz referência a natureza do link:
    • next - indica que o link é prara a próxima parte do documento atual;
    • prev - indica que o link é prara a parte anterior do documento atual;
    • author - indica que é um link para o site do autor do artigo atual;
    • external - indica que um link para outro site que não faz parte do site atual;
    • nofollow - indica que um link para um site não endossado, como por exemplo, um link pago.

O exemplo anterior utilizando a melhor configuração para um link externo, qual seja, target="_blank" e rel="external" fica da seguinte forma:

Links Internos
